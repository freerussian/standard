# standard
«Стандарт свободного русского языка» — открытый свод прескрипций. В этом репозитории фиксируются актуальные принципы стандарта в машиночитаемом виде: спецификация, справочник и словарь. Инструменты проверки текстов на соответствие стандарту и автоматического исправления (линтер и автофикс) публикуются отдельно.
